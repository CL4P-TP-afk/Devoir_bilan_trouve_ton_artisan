# Trouve ton artisan

Projet de formation développeur web.

Plateforme permettant de rechercher des artisans par catégorie
et de les contacter via un formulaire.

## Stack technique
Frontend:
- React
- Bootstrap
- Sass

Backend:
- Node.js
- Express
- MySQL / MariaDB
- Sequelize

## Structure du projet
```
backend/ : API REST
frontend/ : application React
sql/ : scripts base de données
docs/ : documentation du projet
```
